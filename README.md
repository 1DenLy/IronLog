# Gym Voice Tracker Bot 🏋️‍♂️🗣📊

Telegram-бот для hands-free трекинга тренировок. Пользователь надиктовывает прогресс голосом, бот парсит текст и аккуратно раскладывает подходы, повторения и веса по вкладкам в Google Таблицах.

## 🚀 Основной User Flow

1. Пользователь запускает бота, нажимает `/start`.
2. Бот предлагает: «Создать новую таблицу» или «Привязать существующую ссылку».
3. При нажатии **«Новая тренировка»**:
   * Бот создает новый лист (вкладку) в Google Таблице с текущей датой (например, `15.06.2026`).
   * Пишет: *«Тренировка начата! Надиктовывай упражнения в формате: Название упражнения, Вес, Подходы, Повторения»*.
4. Во время тренировки пользователь отправляет голосовые (например: *"Жим лежа 80 килограмм на 10 повторений, сделал 4 подхода"*).
5. Бот:
   * Скачивает `.ogg` файл.
   * Расшифровывает через `faster-whisper`.
   * (Опционально) Структурирует текст через LLM (или регулярные выражения).
   * Дописывает строку в текущий лист таблицы.

---

## 🛠 Технологический стек

* **Language:** Python 3.11+
* **Telegram Bot API:** Aiogram 3 (асинхронный, удобный)
* **STT (Speech-to-Text):** `faster-whisper` (локально, модель `small` или `base`)
* **Google Sheets API:** `gspread` + `google-auth` (работа через Service Account)
* **Парсинг текста:** Регулярные выражения (RegEx) ИЛИ интеграция с легкой локальной LLM для извлечения сущностей (`Упражнение`, `Вес`, `Подходы`, `Повторения`).

## 📦 Структура базы данных / Таблицы

Каждый лист (вкладка) = Одна тренировка.
Структура колонок на листе:

| Время записи | Упражнение | Вес (кг) | Подходы | Повторения | Оригинальный текст |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 18:15 | Жим лежа | 80 | 4 | 10 | "жим лежа 80 килограмм на 10..." |
| 18:32 | Разводка гантелей | 16 | 3 | 12 | "разводка гантелей по 16кг 3 по 12" |
